# Interview Cheating Detection System

AI-powered online interview proctoring platform built using:

- React
- Flask
- OpenCV
- MongoDB
- YOLO

## Features

- Face Detection
- Multiple Face Detection
- Phone Detection
- Tab Switching Detection
- Head Pose Monitoring
- Gaze Tracking
- Violation Reports
- Admin Dashboard

## Tech Stack

Frontend:
- React
- Axios
- React Webcam

Backend:
- Flask
- OpenCV
- MongoDB

AI:
- YOLO
- Computer Vision
